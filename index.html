Welcome to Awesome Xcode Plugins.

This is a Collection of Awesome Xcode Plugins which ease out things needed as a developer

Install Guide

Use Alcatraz, package manager for easier install/un-install of Xcode Plugins

List of Plugins

1. KSImageNamed

Xcode plug-in that provides autocomplete for imageNamed: calls.

2. ColorSenseRainbow

Plugin for Xcode to make working with colors more visual.

3. Lin

Xcode plugin that provides auto-completion for NSLocalizedString..

4. SCXcodeSwitchExpander

Xcode plugin that enables switch cases autocompletion.

5. VVDocumenter

Xcode plug-in which helps you write documentation comment easier, for both Objective-C and Swift.


6. XAlign

An amazing Xcode plugin to align regular code. it can align anything in any way you want.

7. XToDo

Xcode plugin to collect and list the TODO,FIXME.

8. CocoaPods for Xcode

Dependency management helper for your CocoaPods, right in Xcode.

9. HOStringSense for Xcode

Plugin for Xcode to make working with strings less "escaped".
